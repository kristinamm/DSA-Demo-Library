# Binary search trees

```java
// Returns the number of nodes in the BST
int size();

// Returns the depth (height) of the BST
int depth();

// Returns the minimum element of the BST
int nin();

// Prints the tree in pre-order
void printPreorder();

// Prints the tree in-order
void printPreorder();

// Prints the tree in post-order
void printPostorder();

// Returns true if the tree is BST
boolean isBST();
```
